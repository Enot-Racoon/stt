# Speech-to-text browser library

A browser speech-to-text library built with TypeScript and Bun.

## Packages

- `stt-core` - Core speech-to-text engine
- `stt-react` - React hooks and integration
- `stt-ui` - Headless UI components

## Development

```bash
# Install dependencies
bun install

# Run tests
bun test

# Type check
bun run typecheck

# Build all packages
bun run build
```
